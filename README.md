# 2025-jlu-Analog-and-array-logic-circuits-major-project
交通灯设计，应用logisim搭建电路 调试了好久 供参考

题目要求如下：

1 题目要求
1.1 功能场景：
模拟十字路口交通灯（东西方向和南北方向），含紧急按钮（Emergency）、夜间模式按钮（Night）。要求输出两个方向各三种灯和倒计时数码管（显示剩余秒数，为提高要求）：东西方向红/绿/黄灯（EWRed、EWGreen、EWYellow），南北方向红/绿/黄灯（NSRed、NSGreen、NSYellow）。

1.2 亮灯规则：
分为正常、紧急、夜间三种模式，优先级：紧急模式>夜间模式>正常模式。
-  正常模式：东西绿灯亮35s → 东西黄灯亮5s → 东西红灯亮40s ，同时南北红灯亮40s → 南北绿灯亮35s → 南北黄灯亮5s，循环；
- 紧急模式：按下Emergency按钮后，所有方向红灯亮，其他灯灭，倒计时灭；
- 夜间模式：按下Night按钮后，所有方向黄灯闪烁（1秒），其他灯灭，倒计时灭；

1.3 设计要求（设计报告满分90分）：
（1）基本功能（70分）：输入紧急按钮、夜间模式按钮和时钟；输出东西方向红灯、绿灯、黄灯的亮灭，南北方向红灯、绿灯、黄灯的亮灭；组合电路实现灯态控制；时序电路实现正常模式灯态循环；
（2）提高功能（20分）：两个方向用于显示倒计时剩余秒数的两位BCD码数码管设计（使用Logisim中的7段数码管实现）。
